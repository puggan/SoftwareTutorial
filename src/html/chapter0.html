Puggan's Software Developing Tutorial 0 - Hello World

The clasic Hello World program.

Before you can start writing any software, you need to get working envirement, that as a minimum writes a string of text back when executed. The use of the string "Hello World" dates back to a Tutorial from 1974 . In most case a main-function is executed, and it execute a print-command to show the text.

First section bellow is going to show a code-exemple, thats going to be the base for all coming exemples. You can switch between the different language, and see how simular or different they are. After that is instructions for how to execute it, inclusive a short explanation of what the rows do. After that are an installation and envirement setup instruction for each langauge.

Code exemple: Hello World

Javascript, hello.js

                        (async () => {
                            console.log("Hello World");
                        })();

Typescript, hello.ts

                        (async () => {
                            console.log("Hello World");
                        })();

PHP, hello.php

                        <?php
                        (static function() {
                            echo "Hello World", PHP_EOL;
                        })();

Python, hello.py

                        def main():
                                print("Hello World");

                        if __name__ == "__main__":
                                main()

java, hello.java

                        public class Main {
                            public static void main(String[] args) {
                                System.out.println("Hello World");
                            }
                        }

C# (C Sharp), hello.cs

                        using System;

                        public class HelloWorld
                        {
                            public static int Main(string[] args) {
                                Console.WriteLine("Hello World");
                                return 0;
                            }
                        }

C, C99, hello.c

                        #include <stdio.h>

                        int main(char argc, char* argv) {
                                printf("Hello World");
                                return 0;
                        }

Execution exemples

Save the code above in to a file, in the coming examples we going to use the suggested filename from the headers above.

If you installed an IDE for the language you want to test, there may be an Play-icon to run the file.

If you got the tools correctly installed, the examples bellow should work from a terminal, if the file is in the terminals working directory. If you need help installing the tools, check the next section of this page.
